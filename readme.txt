To execute the code:

1-Run the following files: growth.py, industry_migration.py, industry.py, and country_migration.py. These files will generate CSV files that will be used in step 2.
2-Run fact_table.py and industry_dimension.py. These files will generate the dimensions and fact table in CSV files.
3-Run post.py to write the CSV files to the database. Please note that the code in post.py needs to be manually changed for each dimension.
4-The csi4142.sql file contains the database.
